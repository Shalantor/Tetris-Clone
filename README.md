A Tetris clone written in Python 3.5, using Pygame version 1.9.2

Controls:

Left Arrow: Move piece one block left.
Right Arrow: Move piece one block right.
Up Arrow: Change orientation of piece.
Down Arrow: Make piece fall faster.
Space: Make piece instantly fall to final destination marked with the grey color piece at the bottom.
N: Play next soundtrack
M: Mute/Unmute soundtrack
P: Pause/Unpause game

One soundtrack is the original tetris OST, the other 4 are variations/remixes.
